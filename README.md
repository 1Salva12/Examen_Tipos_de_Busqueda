# Examen_Tipos_de_Busqueda

Este repositorio contiene una aplicación Django mínima para desplegar ejemplos de búsqueda.

## Estructura principal

- `manage.py`: comando de administración de Django.
- `examen_tipos/`: proyecto Django.
- `searchapp/`: aplicación con los algoritmos y vistas.
- `requirements.txt`: dependencias necesarias.

## Cómo ejecutar

1. Crear un entorno virtual:

```bash
python -m venv venv
source venv/bin/activate
```

2. Instalar dependencias:

```bash
pip install -r requirements.txt
```

3. Ejecutar el servidor:

```bash
python manage.py runserver
```

4. Abrir en el navegador:

```text
http://127.0.0.1:8000/
```
